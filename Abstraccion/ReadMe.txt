Abstraccion
-----------
1. Interfaces
Es un tipo de clase abstracta que se utiliza para definir metodos que se van a implementar en otras clases.
Declaran solo metodos y constantes, pero no atriubutos.
Se pueden implementar varias Interfases en una Clase.

2. Clases Abstractas
Son clases no instanciables para definir atributos, metodos y metodos abstractos que se van a implementar en otras clases a traves de la herencia.
Crear gerarquias de clase,
Implementar Polimorfismo,
Puede tener atributos, 
Puede tener metodos no abstractos,,
Puede Tenet metodos abstractos,
Puede tener Constructor

ORM: Mapeo de Objetos Relacionales (Hibernate,Eclipselink,Spring)

3. Paralelismo
Procesos multitareas
Utilizado en bach processing
Varios procesos al mismo tiempo
